# Bible Video Maker 사용 설명서

성경 구절을 깔끔한 슬라이드로 만들어 **영상(MP4)** 또는 **이미지(PNG)** 로 저장하는 프로그램입니다.

---

## 1. 시작하기

### 프로그램 실행
- **Mac**: `BibleVideoMaker.app` 더블클릭
- **Windows**: `BibleVideoMaker.exe` 더블클릭

### 화면 구성
프로그램을 열면 화면이 세 부분으로 나뉘어 있어요.

| 위치 | 기능 |
|---|---|
| 왼쪽 | 절 목록 (불러온 성경 구절들) |
| 가운데 | 미리보기 (현재 선택한 절이 어떻게 나오는지) |
| 오른쪽 | 설정 (폰트·색상·레이아웃 등 모든 설정) |

---

## 2. 성경 텍스트 불러오기

### 방법 ① 텍스트 파일 한 개 열기
1. 위쪽 도구 모음에서 **텍스트 열기** 클릭 (또는 `Ctrl+O`)
2. `.txt` 파일 선택

### 방법 ② 두 언어 파일 합치기 (좌우 양쪽에 나오게)
- 왼쪽 절 목록 위의 **두 파일 합치기** 버튼 사용
- 외국어 파일과 한국어 파일을 따로 선택하면 절 번호 기준으로 자동 매칭

### 지원하는 텍스트 형식
프로그램이 자동으로 형식을 감지합니다.

```
1|Hapo mwanzo Mungu...|태초에 하나님이...    ← 좌우 두 언어 (파이프 구분)
1 태초에 하나님이...                         ← 한 언어 + 번호
1:1 태초에 하나님이...                       ← 장:절 형식
태초에 하나님이...                           ← 번호 없이 (자동 번호)
```

## 3. 레이아웃 모드 ⭐ 신기능

설정 패널 → **레이아웃** 탭 → 맨 위 **레이아웃 모드** 드롭다운에서 선택.

### 단일 (한 절씩 가운데)
한 언어만 화면 정중앙에 큼직하게 표시.
한 권의 성경(한국어 또는 외국어 한쪽만)을 시청자에게 보여줄 때 적합합니다.

### 좌우 분할 (외국어 | 한국어)
화면 왼쪽에 외국어, 오른쪽에 한국어. 두 언어를 **나란히** 비교하며 보여줄 때 사용.

### 위아래 분할 (외국어 / 한국어)
화면 위쪽에 외국어, 아래쪽에 한국어. 가로로 긴 텍스트를 **위아래로** 보여주고 싶을 때 사용.

## 4. 출력하기

### 방법 ① MP4 영상으로 저장
- 도구 모음 → **영상 생성 (MP4)** (단축키 `Ctrl+E`)
- 두 가지 모드 중 선택:
  1. **한 개 파일로 합치기** — 모든 절을 이어서 한 편의 영상으로 (기본)
  2. **절마다 별도 파일로 분리** ⭐ 신기능 — 절마다 따로 MP4 파일이 만들어집니다 (편집·SNS용으로 편리)
- 절별 분리 선택 시:
  - 파일명 앞부분(접두사)을 입력 (예: `창세기` → `창세기_001.mp4`, `창세기_002.mp4`, ...)
  - 저장할 폴더 선택
- ffmpeg가 앱에 내장되어 있어 별도 설치 불필요.

### 방법 ② PNG 사진으로 저장 ⭐ 신기능
- 도구 모음 → **이미지 내보내기 (PNG)** (단축키 `Ctrl+Shift+E`)
- 파일명 앞부분(접두사) 입력 (예: `창세기` → `창세기_001.png`, `창세기_002.png`, ...)
- 저장할 폴더 선택
- ffmpeg 필요 없음. 한 장 한 장 그림으로 쓰고 싶을 때 편리합니다.

## 5. 설정 자세히 보기

### 폰트 탭
- 시스템 폰트 선택, 또는 **폰트 파일** 직접 지정 (`.ttf`, `.otf`, `.ttc`)
- 외국어용 / 한국어용 폰트를 따로 지정 가능
- 크기, 굵기, 자간, 행간 조절

### 색상 탭
- 배경, 본문 텍스트, 절 번호, 헤더, 구분선 등 모든 요소 색상 변경

### 레이아웃 탭
- 레이아웃 모드 (위 3번 참고)
- 해상도: 1920×1080 / 2560×1440 / 3840×2160
- 좌우 여백, 컬럼 간격
- 수직(또는 가로) 구분선 표시 여부
- 절 번호 크기·위치
- 텍스트 수직 정렬 (위/가운데/아래)

### 배경 탭
- **기본 (다크 네이비)**: 짙은 남색 + 은은한 빛 효과 (기본값)
- **단색**: 원하는 색 한 가지
- **그라디언트**: 위→아래 두 가지 색이 부드럽게
- **이미지**: 사진을 배경으로. 어둡기 조절 가능

### 헤더 탭
- 화면 맨 위 헤더 텍스트 (예: "MWANZO / 창세기")
- 언어 라벨 (각 컬럼 아래 표시되는 작은 글씨, 예: "KISWAHILI", "한국어")

### 영상 탭
- **절 표시 시간**: 자동(단어 수 기반) / 고정 / 수동
- 페이드 전환 효과 on/off
- FPS (24 / 30 / 60)
- 배경음악 추가 + 볼륨 조절

## 6. 프리셋 저장/불러오기

설정을 다 해놓고 다음에 또 쓰고 싶다면:
- **프리셋 저장** (`Ctrl+S`) → JSON 파일로 저장
- **프리셋 불러오기** → 저장된 JSON 파일 선택

---

## 7. 단축키 모음

| 단축키 | 기능 |
|---|---|
| `Ctrl+O` | 텍스트 파일 열기 |
| `Ctrl+S` | 프리셋 저장 |
| `Ctrl+E` | MP4 영상 생성 |
| `Ctrl+Shift+E` | PNG 이미지 일괄 내보내기 |

> Mac에서는 `Ctrl` 대신 `⌘ (Command)` 키를 쓰세요.

## 8. 자주 묻는 질문

**Q. 절 하나만 PNG로 저장할 수 있나요?**
A. PNG 내보내기는 항상 모든 절을 한꺼번에 저장합니다. 한 절만 필요하면 그 절을 선택한 뒤 미리보기 화면을 캡처하거나, 일단 전부 저장한 후 원하는 파일만 골라 쓰세요.

**Q. 영상 만들 때 ffmpeg 오류가 나요.**
A. ffmpeg가 설치되지 않았거나 PATH에 없는 경우입니다. 설치 후 프로그램을 다시 실행해주세요.

**Q. 한국어 폰트가 깨져요.**
A. 폰트 탭에서 한국어를 지원하는 폰트(예: 본고딕, 나눔고딕 등)를 직접 선택하세요. 폰트 파일을 가지고 있다면 **폰트 파일** 항목에서 직접 지정할 수 있습니다.

**Q. 두 언어 파일의 절 수가 다른데 합쳐도 되나요?**
A. 네. 절 번호로 매칭되며, 한쪽에만 있는 절은 그 쪽만 표시됩니다.
